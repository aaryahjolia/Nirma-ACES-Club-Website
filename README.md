# ACES Club Website, Nirma University

A comprehensive and modern website build for the **Association of Computer Engineering Students (ACES)** at **Nirma University**.

This project was developed for ACES with the goal of creating a dynamic platform to showcase the club's activities, team members, and events. It serves as a central hub for students to connect with the club and stay updated.
